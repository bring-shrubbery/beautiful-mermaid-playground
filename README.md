# Beautiful Mermaid Playground

A browser-based playground for the [beautiful-mermaid](https://github.com/lukilabs/beautiful-mermaid) npm package. Write Mermaid diagram code and see live SVG and ASCII previews.

## Features

- Monaco text editor with live preview
- SVG and ASCII rendering modes
- Dark/light/system theme support
- Copy output to clipboard
- Download SVG as a file

## Getting Started

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- [Next.js 15](https://nextjs.org) (App Router)
- [beautiful-mermaid](https://github.com/lukilabs/beautiful-mermaid) - Mermaid diagram renderer
- [Monaco Editor](https://microsoft.github.io/monaco-editor/)
- [shadcn/ui](https://ui.shadcn.com)
- [Tailwind CSS v4](https://tailwindcss.com)
- [next-themes](https://github.com/pacocoursey/next-themes)
